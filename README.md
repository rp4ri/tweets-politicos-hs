## Description

Este repositorio contiene tweets de tres políticos bolivianos, lo usaré para probar algunos algoritmos de nlp.

El contenido de cada carpeta es el siguiente:

- `data`: Contiene dos carpetas, una para datos sin procesar y otra para datos procesados.
- `figures`: Contiene las figuras que se generan en el análisis.
- `notebooks`: Contiene los notebooks de jupyter que se usan para el análisis.
- `models`: Contiene los modelos generados.
